# 向导与通用智能体

> 面向帮助和对话交互的轻量级智能体。

---

## 向导智能体

处理关于平台的"怎么做"类问题：
- "如何提交训练作业？"
- "有哪些可用的 GPU 类型？"
- "在哪里可以查看我的配额？"

以结构化的帮助信息进行响应：做什么、在哪里做、需要注意什么。具有角色感知能力 — 仅展示当前用户角色（普通用户或管理员）可见的功能。

## 通用智能体

处理问候、闲聊问答以及不属于其他类别的请求：
- "你好" / "你能做什么？"
- 简单的平台问题
- 澄清请求

提供友好的入口。当上下文不足时，避免建议高风险操作或做出假设。

---

两个智能体均为纯 LLM 模式（无工具访问权限）。它们接收用户上下文和能力摘要，但不调用任何工具。
