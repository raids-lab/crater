ACT GPU プラットフォームを使用するには、ACT アカウントが必要です。ACT アカウントをお持ちでない場合は、杜英杰先生（北航新主楼 G512）にお問い合わせください。

## GPU クラスターアクセス権の申請

<Callout>

Slurm GPU クラスターのアクセス権を以前に申請したことがある場合は、このステップをスキップしてください。

</Callout>

[ACT ラボユーザープラットフォーム](https://user.act.buaa.edu.cn/)にログインしてください。ここでは会議室の予約や、GPU クラスターのアクセス権の申請などができます。

まず、右上にあるプロフィールアイコンをクリックしてプラットフォームにログインしてください。

その後、トップナビゲーションバーの「ラボ」をクリックし、「マイ情報」を選択してください。

GPU クラスターのアクセス権をお持ちでない場合、以下のような画面が表示されるはずです：

![GPU クラスターへのアクセス権](./images/no-access-to-gpu.webp)

「使用権限申請」をクリックし、申請フォームを記入し、管理者（杜英杰先生）による審査を待ちます。

## Crater GPU プラットフォームアカウントのアクティベーション

<Callout>

初めてご利用になる場合は、以下の手順でアカウントをアクティベーションしてください。その後は、Crater プラットフォームから直接ログインできます。

</Callout>

アカウントが開通後、「ラボ」->「マイ情報」に新しいGPUクラスターへのアクセスボタンが表示されます。

![新しいGPUクラスターへのアクセス権](./images/new-access-to-gpu.webp)

プラットフォームに初めてログインする際は、このボタンを使ってアカウントをアクティベーションしてください。アクティベーション後、指導教員や研究室などの情報をこのプラットフォームに同期します。

以降のログインでは、ACTアカウントを使ってCraterプラットフォームに直接ログインできます。

## よくある質問

### 1. ラボ外からGPUクラスターにアクセスするにはどうすればいいですか？

クライアントのダウンロード: https://vpn.buaa.edu.cn/portal，
ログイン画面の下にある「クライアントをダウンロード」をクリックし、ポップアップで対応するオペレーティングシステムを選択してダウンロードしてください。

### 2. VPNに接続できない場合の対処法:

ローカルのDNS解析が正しいか確認してください;

```
dig vpn.act.buaa.edu.cn
```

結果がない場合は、「219.224.171.1」を使用して「vpn.act.buaa.edu.cn」にアクセスしてください。
