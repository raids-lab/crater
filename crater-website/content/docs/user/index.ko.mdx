## 플랫폼 개요

Crater 는 북경航空航天대학교에서 독자적으로 개발한 클러스터 관리 플랫폼으로, 사용자에게 효율적이고 편리한 클러스터 자원 관리 솔루션을 제공합니다. 이 플랫폼은 클러스터 내의 계산 자원, 저장 자원 등을 통합적으로 스케줄링하고 관리하여 클러스터의 안정적인 운영과 자원의 최적화된 활용을 보장합니다.

### 직관적인 인터페이스 설계

Crater 는 간결하고 사용하기 쉬운 사용자 인터페이스를 제공하며, 사용자는 그래픽 인터페이스를 통해 다양한 클러스터 관리 작업을 쉽게 수행할 수 있습니다. 예를 들어, 리소스 대시보드를 통해 클러스터 자원 사용 현황 (예: CPU 사용률, 메모리 사용량, 저장 용량 등) 을 실시간으로 확인할 수 있습니다.

작업 관리 인터페이스는 실행 중인 작업, 작업 대기열, 작업 이력 등을 표시하여 사용자가 언제든지 작업 상태를 확인하고 작업을 수행할 수 있도록 합니다.

![jobshow](./assets/jobshow.webp)

### 효율적인 자원 스케줄링

플랫폼은 작업의 우선순위, 자원 요구 사항 등에 따라 작업에 최적의 자원을 자동으로 할당하는 스마트한 자원 스케줄링 알고리즘을 채택합니다. 예를 들어, 여러 작업이 동시에 자원을 요청할 경우, Crater 는 신속히 분석하여 가장 긴박하고 중요한 작업에 자원을 할당하여 전체 실행 효율을 향상시킵니다.

### 모니터링 기능

상세한 모니터링 데이터와 로그 기록은 사용자가 클러스터 운영 상태를 깊이 있게 파악할 수 있는 근거를 제공하며, 사용자가 고장 진단 및 성능 최적화를 수행하는 데 도움을 줍니다.

![log_diagnosis](./assets/log_diagnosis.webp)
