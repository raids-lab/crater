## プラットフォーム概要

Crater は北航が独自に開発したクラスタ管理プラットフォームで、クラスタユーザーに効率的で便利なクラスタリソース管理ソリューションを提供します。このプラットフォームは、クラスタ内の計算リソースやストレージリソースなどを統一してスケジュールし、管理し、クラスタの安定した運用とリソースの最適な利用を確保します。

### 直感的なインターフェース設計

Crater はシンプルで使いやすいユーザーインターフェースを提供しており、ユーザーはグラフィカルな操作インターフェースを通じて、さまざまなクラスタ管理タスクを簡単に実行できます。たとえば、リソースダッシュボードは、クラスタリソースの使用状況をリアルタイムで表示します。このダッシュボードでは、CPU 使用率、メモリ使用量、ストレージ容量などの指標が表示されます。

ジョブ管理インターフェースでは、実行中のジョブ、ジョブキュー、ジョブ履歴が表示され、ユーザーはいつでもタスクの状態を確認し、操作を行うことができます。

![jobshow](./assets/jobshow.webp)

### 高効率なリソーススケジューリング

プラットフォームは、ジョブの優先度、リソース要件などの要因に基づいて、スマートなリソーススケジューリングアルゴリズムを採用しており、ジョブに最適なリソースを自動的に割り当てます。たとえば、複数のジョブが同時にリソースを要求する場合、Crater は迅速に分析し、最も緊急で重要であるタスクにリソースを割り当て、全体の実行効率を向上させます。

### モニタリング機能

詳細なモニタリングデータとログ記録は、ユーザーにとってクラスタの運営状況を深く理解するための根拠を提供し、障害の特定と性能最適化が容易になります。

![log_diagnosis](./assets/log_diagnosis.webp)
