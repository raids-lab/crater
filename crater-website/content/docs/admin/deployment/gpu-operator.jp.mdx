## 概要

**NVIDIA GPU Operator** は、Kubernetes クラスターで GPU をサポートするために必要なすべてのコンポーネントの展開および管理を自動化します。

Crater では以下を提供します：

- GPU ドライバのインストール  
- NVIDIA コンテナランタイムの設定  
- GPU 監視用の `dcgm-exporter`（Prometheus スタックで使用）  
- Crater のジョブスケジューリングおよび GPU メトリクス表示とのスムーズな統合

> Crater は GPU Operator を必要とし、GPU ジョブが正しくスケジュールされ、監視されることを保証します。

---
